Quick Start

Five minutes to your first policy.

$ npm install @paysentry/core @paysentry/observe @paysentry/control
TypeScript 
import { SpendTracker } from '@paysentry/observe';
import { PolicyEngine, dailyLimit } from '@paysentry/control';

// 1. Create a spend tracker for your agent
const tracker = new SpendTracker({ agentId: 'agent-001' });

// 2. Define a policy: $100/day max
const policy = new PolicyEngine([
  dailyLimit({ amount: 100, currency: 'USD' }),
]);

// 3. Evaluate a transaction before execution
const decision = await policy.evaluate({
  agent: 'agent-001',
  amount: 25.00,
  recipient: 'api.example.com',
  protocol: 'x402',
});

// 4. Get spend analytics
const stats = await tracker.analytics({ period: '24h' });
console.log(stats.totalSpend, stats.txCount, stats.topRecipients);
